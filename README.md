### Installation T-Remix
* `Commands` for termux
```
$ termux-setup-storage
  
$ pkg update && pkg upgrade && pkg install git -y

$ git clone https://github.com/DNReporters/DNloginbanner

$ ls

$ cd DNloginbanner

$ ls

$ chmod +x *

$ bash DNloginbanner.sh
```



### To Remove Banner
* `Commands` for termux
```
$ cd DNloginbanner

$ ls

$ bash remove-banner.sh
```
### To Remove Login
* `Commands` for termux
```
$ cd T-Remix

$ ls

$ bash remove-login.sh
```
